
# نظام محاسبة مكتبي يعتمد على Excel

هذا مشروع مكتبي عربي بسيط لإدارة البيانات المحاسبية المستوردة من **اليومية الأمريكية** إلى قاعدة بيانات **SQLite**، مع واجهات تشغيل محلية على سطح المكتب.

## الوحدات الموجودة
- لوحة التحكم
- استيراد اليومية الأمريكية
- دفتر اليومية
- المدينون المتنوعون
- أذون الصرف الذكية
- الأستاذ العام
- ميزان المراجعة

## الملفات
- `main.py` : تشغيل الواجهة
- `db.py` : قاعدة البيانات
- `importer.py` : استيراد ملف Excel
- `services.py` : التقارير والاستعلامات
- `accounting.db` : قاعدة البيانات المستوردة مسبقاً من الملف المرفوع

## طريقة التشغيل
```bash
pip install -r requirements.txt
python main.py
```

## إنشاء ملف exe
```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name SmartAccounting main.py
```

## ملاحظات
- تم استيراد الملف `يومية امريكية 2022 12-3.xlsx` داخل قاعدة البيانات المرفقة.
- عند استيراد ملف جديد سيتم إعادة بناء الحسابات والقيود من جديد.
- أذون الصرف الذكية في هذه النسخة تستخرج الحركات التي يخرج فيها المبلغ من بنك/خزينة إلى حساب مستفيد.
- يمكن لاحقاً تطوير النظام إلى نسخة Electron أو إضافة تقارير PDF وطباعة فواتير ومراكز تكلفة.
